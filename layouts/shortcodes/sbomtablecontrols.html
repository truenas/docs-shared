Open the drop down and select a product version to view a simplified SBOM for that version. An SBOM can take a moment to load.

When an SBOM is loaded, click Download SBOM to download the full SBOM as a Software Package Data Exchange (SPDX) formatted .json file. Filter the table with keywords in the Search box. Reset Table unloads the selected SBOM.

Clicking a table header sorts entries in ascending or descending alphabetical order.
